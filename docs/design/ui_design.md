# UI/UX設計書

## 1. デザイン原則
- **モバイルファースト**: スマートフォン優先の設計
- **レスポンシブ対応**: PC、タブレット、スマートフォン対応
- **ユニバーサルデザイン**: 高齢者や障害者にも使いやすい設計
  - 最小文字サイズ: 16px
  - 行間: 1.6倍
  - コントラスト比: WCAG AA準拠
  - タップ領域: 最小44×44px

## 2. 画面一覧
1. ログイン画面
2. ダッシュボード
3. 納品書管理
   - 一覧画面
   - 画像アップロード画面
   - 認識結果確認画面
   - 詳細画面
   - 編集画面
4. 請求書管理
   - 販売員請求書一覧
   - 委託先請求書一覧
   - 詳細画面
5. マスタ管理
   - 販売員一覧
   - 商品一覧
   - 委託先一覧
6. 領収管理画面
7. 設定画面

## 3. 主要画面詳細

### ログイン画面
- ユーザー名/パスワード入力フィールド
- ログインボタン
- エラーメッセージ表示領域
- パスワード忘れリンク（将来拡張用）

### ダッシュボード
- KPIカード: 当月納品書件数、未請求金額、未収金額
- 最近の活動リスト
- 月次売上グラフ
- クイックアクセスボタン

### 納品書一覧画面
- 検索バー
- フィルタ（販売員、日付範囲）
- ソートオプション
- ページネーション
- 各行: 納品書番号、日付、販売員、金額、ステータス

### マスタ管理画面（例: 商品一覧）
- 一覧テーブル（名前、価格、フラグ、表示順）
- 検索・ソート機能
- 追加ボタン
- 編集・削除アクション

## 4. ナビゲーション
- サイドバーまたはボトムナビゲーション
- メニュー項目:
  - ダッシュボード
  - 納品書管理
  - 請求書管理
  - マスタ管理
  - 領収管理
  - 設定

## 5. コンポーネント設計
- **ボタン**: プライマリ、セカンダリ、危険ボタン
- **フォーム**: テキスト入力、数値入力、日付ピッカー、オートコンプリート
- **テーブル**: ソート可能、ページネーション付き
- **モーダル**: 確認ダイアログ、詳細表示
- **通知**: トーストメッセージ

## 6. カラースキーム
- プライマリ: 青系 (#3B82F6)
- セカンダリ: グレー系 (#6B7280)
- アクセント: 緑系 (成功)、赤系 (エラー)
- 背景: 白/ライトグレー

## 7. タイポグラフィ
- フォント: システムフォント（San Francisco, Roboto等）
- サイズ: 16px (本文), 18px (見出し), 24px (タイトル)
- 行間: 1.6

## 8. インタラクション
- ホバー効果
- クリックアニメーション
- ローディング状態表示
- エラー状態の視覚フィードバック
